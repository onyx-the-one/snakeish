# snakeish
Just a snake game, nothing more rlly.

But okay, theres some config options in the beginning, defaults are:
  Resolution 1200x800px with 20px squares
  8 FPS speed with .1 increments on food
  5 obstacles generated random
  Wrap turned on
You can modifiy any of these, theres also my high score counter so add yours ;w;

Do what you want, your life is yours. Wing it. -onyx
